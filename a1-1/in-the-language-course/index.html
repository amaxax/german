# Sprache im Kurs

# Wie heißt das auf Deutsch?

  • der Stuhl
  • die Kaffeemaschine
  • das Handy
  • die Stühle

# Was ist das?

  • Das ist ein Stuhl.
    • Nein. Das ist kein Stuhl.
  • Das ist eine Kaffeemaschine.
    • Nein. Das ist keine Kaffeemaschine.
  • Das ist ein Handy.
    • Nein. Das ist kein Handy.
  • Das sind keine Stühle.
    • Nein. Das ist kein Stühle.

# Ist das eine Kamera?

Nein, das ist keine Kamera. Das ist eine Lampe.

# Sind das Radios?

Nein, das sind keine Radios. Das sine Lampen.
